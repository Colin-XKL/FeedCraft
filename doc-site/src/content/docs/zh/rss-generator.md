---
title: RSS 生成器
description: 使用可视化的 RSS 生成器工具将任何网站转换为 RSS 订阅源。
---

**RSS 生成器 (RSS Generator)** 是 FeedCraft 内置的一个强大工具，允许您将几乎任何网站转换为 RSS 订阅源。它提供了一个可视化界面，让您可以在网页上选择元素，并将它们映射到 RSS 字段，如标题、链接、日期和内容。

## 如何访问

您可以在 FeedCraft 仪表板的 **工具 (Tools)** > **RSS 生成器 (RSS Generator)** 下找到此功能。

## 使用流程

### 1. 获取页面

1.  输入您想要转换为 RSS 的网站 URL。
2.  点击 **Fetch Page** (获取页面)。
3.  页面内容将加载到左侧的“页面预览”区域。

:::note
该工具使用后端代理来获取页面内容，可以处理一些基本的反爬虫保护。但是，极其复杂的动态网站可能仍需要更高级的配置。
:::

### 2. 选择模式

页面加载后，您可以使用 **Selection Mode** (选择模式) 来可视化地选取元素。

-   **切换模式**：点击 "Selection Mode" / "Preview Mode" 按钮，在与页面交互（预览）和选取元素（选择）之间切换。
-   **可视化选取**：在选择模式下，鼠标悬停在元素上会高亮显示。点击元素将把该元素的选择器填入当前激活的字段。

### 3. 配置选择器

配置过程分为两个主要步骤：

#### 第一步：定义列表项 (List Item)
首先，您需要确定代表单个文章或项目的重复元素。

1.  点击 "List Item Selector" 输入框里的 **Pick** 按钮。
2.  在预览区域中，点击其中一个文章卡片或列表项。
3.  工具将自动生成一个 CSS 选择器（例如 `.article-card`），并显示匹配了多少个项目。

#### 第二步：映射字段
定义好列表项后，您可以映射相对于每个项目的具体字段。

-   **Title Selector (标题)**：点击 **Pick**，然后在列表项中选择标题元素。
-   **Link Selector (链接)**：点击 **Pick**，然后选择链接元素（通常是标题或“阅读更多”按钮）。
-   **Date Selector (日期)**：(可选) 选择日期元素。
-   **Content Selector (内容)**：(可选) 选择摘要或内容元素。

:::tip[键盘快捷键]
在选择模式下，您可以使用 **向上箭头** 和 **向下箭头** 键在 DOM 树中导航，精确选择父元素或子元素。
:::

### 4. 预览 RSS 项目

配置好选择器后：

1.  点击 **Preview RSS Items** (预览 RSS 项目)。
2.  右侧的“Extracted Items” (提取的项目) 面板将显示从页面提取的数据。
3.  验证标题、链接和其他字段是否正确。

## 高级用法

-   **手动 CSS 选择器**：如果自动生成的选择器不够精确，您可以手动输入或编辑 CSS 选择器。
-   **相对选择**：标题、链接、日期和内容选择器是相对于列表项选择器的 *相对路径*。这确保了数据能够正确地归类到每个项目中。

## 故障排除

-   **获取失败 (Fetch Failed)**：如果页面加载失败，请检查 URL 是否正确。某些网站可能会阻止抓取工具。
-   **未找到项目 (No Items Found)**：仔细检查您的“列表项选择器”。确保它与页面的 HTML 结构匹配。
-   **数据错误**：如果标题或链接不正确，请尝试重新选择特定元素，或使用箭头键选择父容器。
